# walrusBox

**walrusBox** is a lightweight and easy-to-use **horizontal content slider** (for images, news, etc.), originally developed as a jQuery plugin, now available in pure JavaScript. I recreated it because I believe that the forgotten old can be something new and useful.

---

## Features

- Horizontal sliding of content  
- Automatic slide change at a specified interval  
- Navigation triggers (dots) for manual slide selection  
- Responsive to container width  
- Picture frame added in JS version

---

## Installation

1. Copy the files into your project.  
2. Place your images in a folder, e.g., `img/`.  
3. Include the script in your HTML:

```html
<script src="js/walrusBox.js"></script>
``` 
| Parameter   | Description                           | Example  |
| ----------- | ------------------------------------- | -------- |
| `amount`    | Number of slides                      | `4`      |
| `interval`  | Slide change interval in milliseconds | `3000`   |
| `direction` | Slide direction (`left` or `right`)   | `'left'` |
